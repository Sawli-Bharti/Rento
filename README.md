# Rento
Rento is a full-stack rental management platform designed to digitize the entire rental lifecycle for tenants and property owners.

The system allows tenants to browse properties, schedule visits, and request bookings. Property owners can review and approve or reject booking requests. Once approved, tenants can securely pay the security deposit through Razorpay, after which a rental agreement is created.

The platform also provides a rental dashboard where tenants can track their rented properties, pay monthly rent, and view payment history. Owners can monitor bookings, active rentals, and tenant activity.

Key Features:
- Property listing and discovery
- Visit scheduling and booking requests
- Owner-based booking approval system
- Razorpay payment integration (deposit & monthly rent)
- Rental agreement management
- Monthly rent tracking and payment history
- Role-based dashboards (Tenant & Owner)

Tech Stack:
- Backend: Spring Boot, JPA, MySQL
- Frontend: React.js
- Payment Gateway: Razorpay (Test Mode)
- Architecture: RESTful APIs with DTO projection optimization

Features: Booking | Payments | Rental Tracking | Owner Approval  
Security: JWT Authentication + Payment Verification  
Payments: Razorpay Integration (Test Mode)

This project demonstrates real-world backend design, secure payment handling, and scalable full-stack architecture similar to production rental platforms.
